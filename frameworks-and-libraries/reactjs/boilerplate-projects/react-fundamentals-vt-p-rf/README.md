## Overview
- Author Quiz application to demonstrate react fundamentals
- Concepts covered in this project
	- Components
		- Defining and rendering components
		- Populating props
		- Composing components
		- State
		- Default props
		- Validating props
	- JSX
	- Events
		- DOM events
		- Component events
	- Forms
		- Refs
		- getDOMNode
	- Routing


## Server deployment

### Install npm packages

        npm install

### Install bower packages

        bower install

### Start the server

        grunt serve
